=== TRAITS & TAGS REFERENCE (UPDATED) ===

[ LOGIC TAGS ]
Digunakan dalam array `tags: [...]` di data.js.

--- Role & Behavior ---
win_condition   : Unit yang fokus menghancurkan tower (Hog, Giant, Balloon).
tank            : HP tinggi, pelindung pasukan di belakang (Golem, PEKKA).
mini_tank       : HP menengah, semi-tank (Valkyrie, Knight).
dps             : Damage per second tinggi (Musketeer, Hunter).
glass_cannon    : Damage besar tapi HP sangat tipis (Princess).
swarm           : Unit bergerombol banyak (Skeleton Army, Minion Horde).
aoe             : Area of Effect, menyerang banyak musuh (Wizard, Valkyrie).
support         : Memberikan buff/heal atau serangan jarak jauh (Healer).
control         : Mengganggu gerakan musuh (Ice Wiz, E-Wiz, Tornado).
cycle           : Biaya elixir rendah untuk memutar deck (Ice Spirit, Skeletons).
spawner         : Memunculkan unit lain secara berkala (Witch, Tombstone).
defense         : Bangunan pertahanan (Cannon, Tesla).
siege           : Bangunan yang bisa menyerang tower dari wilayah sendiri (X-Bow).

--- Fisik & Gerakan ---
ground          : Unit berjalan di darat.
air             : Unit terbang (melewati sungai & bangunan).
heavy           : Unit berat, tidak mudah didorong (PEKKA, Golem, Giant).
fast            : Unit bergerak sangat cepat (Hog, Lumberjack).
river-jumper    : Unit darat yang bisa melompati sungai (Hog, Prince).
building-hunter : Prioritas menyerang bangunan saja.
kamikaze        : Mati saat menyerang (Spirits, Wall Breakers).
shielded        : Memiliki shield/nyawa cadangan (Dark Prince, Guards).

--- Special Effects ---
healer          : Unit yang bisa menyembuhkan (Battle Healer).
stun-effect     : Serangan memberikan efek Stun (E-Wiz).
slow-effect     : Serangan memberikan efek Slow (Ice Wiz, Ice Golem).
ramp-damage     : Damage meningkat seiring waktu (Inferno Tower/Dragon).
hide-when-idle  : Masuk ke tanah saat tidak ada musuh (Tesla).


[ STATS PROPERTIES ]
Digunakan dalam objek `stats: { ... }`

hp              : Hitpoints (Nyawa).
shield          : Nyawa tameng (opsional).
dmg             : Damage per hit.
hitSpeed        : Kecepatan serangan (detik).
speed           : Kecepatan gerak (0.5 lambat - 2.0 cepat).
range           : Jarak serangan (grid). 0 = Melee.
sightRange      : Jarak pandang untuk mengejar musuh.
deployTime      : Waktu tunggu saat unit diletakkan (detik).
splashRadius    : Radius area damage (grid).
multiTarget     : Jumlah target yang diserang sekaligus (E-Wiz = 2).
count           : Jumlah unit yang di-spawn (Skeleton x3).

--- Advanced Targeting ---
targetType      : 
    'ground-only' : Hanya serang darat (Knight, Bomber).
    'ground-air'  : Serang darat & udara (Archer, Wizard).
    'air-only'    : Hanya serang udara (Jarang).
    'allies-only' : Hanya target teman (Healer, Drummer).

targetPreferences : Array tag prioritas. (Contoh: ['tank', 'win_condition'])
                    Unit support akan memilih target dengan tag ini meski jauh.

--- Melee System ---
meleeType       :
    'single'      : Serangan ke 1 target (PEKKA).
    'circular'    : Serangan area 360 derajat (Valkyrie).
    'cleave'      : Serangan area kerucut di DEPAN unit (Dark Prince, MK).

--- Projectile System ---
projectile      : Objek konfigurasi `{ type: '...', speed: ..., visual: ... }`
    type        : 'normal', 'boomerang', 'rolling', 'rolling_log', 'spear'.
    visual      : 'beam' (Laser), 'lightning' (Petir Instan).
    count       : Jumlah proyektil per tembakan (Hunter x5).
    spread      : Sebaran proyektil (Hunter spread 1.5).


[ MODULAR EFFECTS SYSTEM ]
Digunakan dalam objek `effects: { ... }` di data.js.

Trigger Keys:
  onHit         : Efek saat serangan mengenai target.
  onSpawn       : Efek saat unit selesai deploy.
  onDeath       : Efek saat unit mati.
  aura          : Efek area pasif di sekitar unit.

Properties per Effect:
  type          : 'damage', 'heal', 'stun', 'slow', 'spawn', 'spell', 'rage'.
  amount        : Nilai damage/heal/boost.
  duration      : Durasi efek (detik).
  radius        : Radius area efek (grid).
  target        : 'enemy' (default), 'ally', 'self'.
  visual        : Override visual (misal: 'poison', 'zap').

  -- Khusus type: 'spawn' --
  unit          : Key unit yang dimunculkan.
  count         : Jumlah unit.

  -- Khusus type: 'spell' --
  spell         : Key spell yang dipicu (misal: 'fireball').
  delay         : Waktu tunggu (detik).


[ ABILITIES ]
Digunakan dalam objek `abilities: { ... }`

charge: {       (Prince, Dark Prince)
    speedMult: 2.0,  // Pengali kecepatan
    dmg: 600,        // Damage saat tabrak
    windup: 90       // Frame persiapan
}

jumpAttack: {   (Mega Knight)
    minRange: 3.5,
    maxRange: 5.0,
    dmg: 444,
    speed: 1.5,
    radius: 2.5
}
